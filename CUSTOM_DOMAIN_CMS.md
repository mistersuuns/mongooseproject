# Custom Domain CMS Access

## Question 1: CMS URL with Custom Domain

**Yes!** Once you set up your custom domain, the CMS will be accessible at:

```
https://yourcustomdomain.com/admin/
```

**How it works:**
- Netlify automatically serves `/admin/` from your custom domain
- No configuration needed
- Just set up the custom domain in Netlify settings

**Setup:**
1. Go to: Netlify Site Settings â†’ Domain Management
2. Add your custom domain
3. Update DNS records
4. CMS will work at: `https://yourcustomdomain.com/admin/`
